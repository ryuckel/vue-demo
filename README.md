# vue-demo

## Vue Project 用フロントエンド ライブラリ検証リポジトリ

検証したいリポジトリをブランチを切って検証していきます。

## ライブラリ一覧

- tui-image-editor
  画像編集系の機能を提供しているライブラリの検証ブランチ
- vue-image-lightbox
  画像のスライドショーを作成するための機能を提供しているライブラリの検証ブランチ

## Project setup

```
yarn install
```
